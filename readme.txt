== Description ==
Sendmsg Elementor Addon adds the ability so you can add the widget in your page and can select the email list and will populate all the custom fields from where you can select the required one and map with the Elementor forms.

You can select the action in the Elementor forms and select Sendmsg from the dropdown and in the next step, it will allow you to select list and custom fields which you can map very easily.

In the Sendmsg platform you can perform various actions some of them are :

⦿	A personal message to each customer
⦿	Sending an email as a journal entry
⦿	Send an email with one click to people who have not opened it before
⦿	Flexibility changes in designs is high and fun
⦿	Exact graph of when it is best to send an email to your customers
⦿	Matching emails to mobile 100%
⦿	Change HTML code in seconds
⦿	Open photos in emails and attachments


== Installation ==
Upload the entire folder to the /wp-content/plugins/ directory.
Activate the plugin through the ‘Plugins’ menu in WordPress.
Go to the Settings menu under this, there will be the Sendmsg (Settings>Sendmsg).
Configure the settings by putting the Site ID and API key.

== Frequently Asked Questions ==
= From where I can get site ID and API key? =

You can generate the API key and Site ID by going here : 

https://panel.sendmsg.co.il/default.aspx?lang=Eng

Once done, you can add the details in Settings > Sendmsg
